旨在少用鼠标/触摸板，避免分心。但是触摸板在快速定位，滚动屏幕和选中非完整行上比较方便。

设置快捷键：`Ctrl + K, Ctrl + S`

<!--more-->

## 外观

打开关闭主侧边栏：`Ctrl + B`

打开关闭下方面板：`Ctrl + J`

打开侧边预览：`Ctrl + K, V`

进入退出禅模式：`Ctrl + K, Z`

切换选项卡：`Ctrl + Tab`

## 光标与选择

选择：`Shift + 方向键`

快速选中一行：`Ctrl + L`

向下选中多行：继续按`Ctrl + L`

选中多行：光标移动到行首，然后`Shift + 上下`

光标移动到行首：`Fn + 左`

从光标处选中到行首：`Shift + Fn + 左`

按单词移动光标：`Ctrl + 左右`

滚动屏幕，光标处于原位置：`Ctrl + 上下`

## 编辑

格式化文档：`Shift + ALt + F`

注释/取消注释：`Ctrl + /`

快速复制一行：`Shift + Alt + 上下`

移动行：`Alt + 上下`

按行号跳转：`Ctrl + G`

快速删除一行：`Ctrl + Shift + K`

删除一行并保留空行：`Fn + 右, Shift + Fn + 左 , Backspace`

## 查找替换

查找框：`Ctrl + F`

查找框上的按钮：

- 区分大小写：`Alt + C`

- 全字匹配：`Alt + W`

- 正则表达式：`Alt + R`

- 下一个：`Enter`

- 上一个：`Shift + Enter`

- 在选定内容中查找：`Alt + L`

替换框：`Ctrl + H`

替换框上的按钮：

- 替换：`Enter`

- 全部替换：`Shift + Enter`

关闭查找替换框：`Esc`
