## MMPI

统计学家 George Box 说：所有模型都是错的，有些是有用的。

这里研究的是对于明尼苏达多相人格测验（Minnesota Multiphasic Personality Inventory）的一系列判断题，回答【是】或【否】对最终结果的影响。依据是[知乎网友 aweeee](https://zhuanlan.zhihu.com/p/600297840)分享的 `xlsx` 文件。

1. 做一道判断题，回答是或否
2. 答完一道题之后，对某一个或者某几个量表加 1 分或不加分
3. 答完所有题之后，会得到十四个分，包括四个效度量表（Q L F K）和十个临床量表（Hs D Hy Pd Mf Pa Pt Sc Ma Si）。这十四个分称作【原始分】。
   - 如果你做的是 566 道题的，额外会有五个附加量表（MAS Dy Do Re Cn）。笔者做的都是两百道题的，只有这十四个分。有一种说法是：在测试系统的后台里有上百个加分项，而打印出来的只有一部分。所以想作弊是很难的——**如果你被别人认为“有病”，且“作弊”成功了，那正好证明你“没病”**。或者如果你想伪装成精神病人，来减轻你犯下的罪行：这个前提就错了，如果精神病人和正常人的区分标准是，控制自己的行为不对别人造成负面影响的能力，那么如果你杀人了，说明你控制不住自己的怒气，说明你本来就可以被定义成精神病人，那就不存在伪装这一说了。就是所谓的 **“论迹不论心”**。
4. 特别地，对这五个临床量表加上一定比例的 K 分，加完之后的称为【K 校正分】

   - Hs += 0.5K
   - Pd += 0.4K
   - Pt += 1.0K
   - Sc += 1.0K
   - Ma += 0.2K

5. 现在我们有十四个分。对除 Q 之外的十三个分分别计算 T 分：

   $$
   T = 50 + \frac{10 \times (你的得分 - 常模均值)}{常模标准差}
   $$

   其中，【常模均值】和【常模标准差】由你的国籍、性别和年龄范围选择。

   【常模均值】是一群正常人（健康人）做的平均分。用你的得分与其相减，刻画的是你与正常人的偏离程度。

   【常模标准差】刻画的是这一群正常人得分的离散程度。用其作分母，可以缩小误差。

四个效度量表，测你做的结果是否有效：

### Q（Question）【疑问】：你没有回答的题目数

在实际测验中，与“性”有关的几个题目可以选择跳过回答。

### L（Lie）【说谎】：你往圣人方向靠的程度

例如

> 做游戏的时候，我只愿赢而不愿输

> 偶尔我会想到一些坏得说不出口的事

> 有时我也讲假话

等的这一些问题，回答【否】会加分

### F（Validity）【诈病】：你装病的程度，或者判断力丧失的程度

例如

> 有人一直想毒死我

> 有时我觉得有鬼神附在我身上

> 我能在我周围看到其他人所看不到的东西、动物和人

等的这一些问题，回答【是】会加分

### K（Correction）【校正】：你对自己的私事隐瞒的程度、防御的程度

例如

> 当我正在做一件重要事情的时候，如果有人向我请教或打扰我，我会不耐烦的

> 我觉得许多人喜欢夸大自己的不幸，以便得到别人的同情和帮助

> 我不在乎别人对我有什么看法

回答【否】会加分

> 我几乎没有和家里人吵过嘴

回答【是】会加分

**十个临床量表，测你病的严重程度：**

### Hs（Hypochondriasis）【疑病】：你对自己身体过度关心的程度

例如

> 我的胃口很好

> 我睡得不安，容易被惊醒

> 我的身体和我的大多数朋友一样地健康

> 我常觉得头上好象有一根绷得紧紧的带子

> 我不容易疲倦

> 有时候我觉得我的头顶一碰就疼

问的都是与你身体相关的，如果你给的反常回答太多了：

- 如果你说的是实话：说明你确实有这些症状，说明你很反常
- 如果你说的是中话：说明你在过度关心得病，说明你很反常
- 如果你说的是假话：说明你在诉苦，说明你很反常

## 自我指涉

自我指涉是什么意思？

你可能会注意到，这个网页的名字叫 `this.html`

刘慈欣在《三体》第二部里写到，写小说的人是作为一个**观察者**，记录下他脑子里上演的东西。

类似地，笔者在这个环境里，营造的是一个虚假的自我、理想的自我。实际上笔者**把时间都花在打字上了**。为了打出这一个标题下面的字，可能会花费好几个小时。

进而，对于笔者打出的有些字，是**通过前面的语境续写的**，而不一定是笔者真实经历的。

有一首歌叫[《不完备定理》](https://music.163.com/#/song?id=2030238327)，笔者是有类似的想法的，但是根本达不到那个境界——这是人家用实力写出来的歌，不是你写的。就像，你读了一本《名人传》，就能认为你是名人吗。

专辑的名字叫《文理叙术 The Contextual Order》，意思是【上下文顺序】。曲风属于【前卫摇滚】——在笔者看到的互联网语境下，这不是一个好名词。这跟吗啡差不多，在紧急状态之下可以镇痛，滥用就变成毒品了。

写小说，前提是你之前读过非常多的文学作品。像笔者这样，属于前言不搭后语。

写论文为什么要用【笔者/我们】，而不是【我】？

如果用后者：一是，会显得你很自私。二是，读你东西的人，会代入不了语境。

## 思维反刍的成因

思维反刍的意思是：一直重复地想同一件事

因为你想得到一个结果，但是一直没有得到。

比如：你在想你之前的不愉快经历。这是因为你想得到一个更好的结果，所以在脑子里一直重演。

又比如：你之前在互联网上发了一些东西，有很多人回复你。但是，【发出东西】这个行为是通过【想】+【动动手指】实现的，并没有经历【通过声带说话】这一步。

这就造成一种条件反射：只要想了，就有人回复你了

所以，当没有人回复你的时候，你就会一直想。

这和使用搜索引擎/点视频看差不多。打几个字/点进去，立马就得到了。当你想睡觉的时候，放下手机——麻了。

笔者父亲小时候也有类似的经历。随着之后的工作和生活，他的**注意力被转移了**。现在回想起来，这是因为他太孤单了。

## 关于边听歌边睡觉，直至入眠

这是因为，你的注意力**集中**在了音乐身上，你就不会想别的事。

但是——如果你听到一个量了，你可能会觉得，之前、未来听的一些歌没有意思了，就是吃菜吃腻了。你的注意力会继续**发散**。

你脑子里有一个缓冲区，你可以通过这个缓冲区，预测到歌曲下一步的发展。

当歌曲下一步的发展，与你预测的有一定区别时，你才会觉得歌曲有一种新鲜感。

对于睡觉，一直不新鲜、一直新鲜不一定是好事。因人而异

——这是一种极其错误的说法。就跟你读了几本小说，就认为世界上所有小说都没有意思了一样。

还是听得太少。

## 精神分析与认知行为疗法的区别

按弗洛伊德的精神分析，就是甩锅给过去的别人和自己。

- 如果甩给别人：将心比心，别人甩给你你愿意接吗？
- 如果甩给自己：被批评者为了逃避问题会变成批评者，批评者批评完了，被批评者还是那样，话都让批评者说了。本我和超我向两边拉，自我向上克服不了重力，五力平衡，合力为零。

认知行为疗法是把每一个正常人都有的能力抽了出来，用来对付精神病人：

> （这件事）对你来说意味着什么？<br/>
> （如果你预想的是真实的）将会发生什么？

大音希声。它关注情绪、思想和行为的互相影响。

## 能指和所指

能指就是符号，所指就是符号所指向的意义。

能指是【生】，所指是 `birth` `biology` `life` `rusty` `strange` `-er` `-ist` ……

能指是变量名，所指是变量值。

变量名可以被重新赋值，就是老子说的“非常名”。

简单点说就是【一词多义】【古今异义】【结合上下文理解】

### C 语言中的指针

#### 普通变量

```c
int num = 100;
```

变量类型`int`，变量名`num`->变量值`100`。

普通变量名已经有一层指向了。

#### 指针变量

用于专门存放内存地址（值的位置）的变量，叫指针。

```c
int* ptr = &num;
```

变量类型`int*`，变量名`ptr`->变量值`num->100的内存地址`

（这种说法好像是不严谨的，`int*`好像是 C++ 的说法）

`&num`，就是获取`num`值的位置。

- `&`一个变量名，就是获取变量名的值的地址。简称取地址。
- `*`一个变量名，就是获取变量名的值（只对指针有效）。简称访问地址。
- `*&`一个变量名就是先`&`后`*`（两个运算符优先级相同，且都是右结合）。
  - 先取地址，后访问地址，和单独使用变量名效果一样。
- `&`一个指针变量名，就是获取指针变量值本身的位置（存放`num`地址值的地址）。

还可以创建二级指针，就是指向指针的指针：

```c
int** pptr = &ptr;
```

- `pptr`变量名指向某一个地址，这个地址处记录着`ptr`的地址
- `ptr`变量名指向某一个地址，这个地址处记录着`num`的地址
- `num`变量名指向某一个地址，这个地址处记录着`num`的值`100`

这样你连拉枪的意识都没了。就算有意识，在近身作战的时候，你拉的同时算法也在拉。你和算法在互相争夺枪的控制权：你手指在触摸屏上移动的距离，和准星实际移动的距离不是线性关系，导致你永远掌握不了那个度。

#### 找掩体、卡视角

和平精英分第三人称模式和第一人称模式。我们一般玩的是第三人称模式，就是可以看到你这个人的身子。第一人称只能看到你真正应该看到的东西。

在第三人称模式下，信息是不对等的：你可以在一个墙角卡视角，看到侧面过来的人，但是对方看不到你。

这在团队竞技模式下非常有用。团竞地图本来就小，4v4。所以团队竞技又叫卡点竞技。

野外作战的时候，拿反斜坡、石头、树或者载具作掩体是生存的基本思路。还有烟雾弹不是摆设，除了救人，跑圈转移的时候也可以用，这是在你位置已经暴露或者有很大风险暴露的情况下。

#### 团队竞技要有点团队意识

一般来说，随机匹配的队友都不会开麦，就算开麦也没人指挥、报点，各打各的。

这时你可以有点团队意识：跟在一个队友后面捡漏。

#### 进圈要紧

现在到最后几个圈了，你还在毒里，向安全区跑。这时你看到五十米开外有个人也在跑圈，他没看见你，请问你打不打？

如果打起来了，活下来的一方一是浪费时间，二负伤，三是暴露位置给圈里的人了。你们都活不了。

还有一个套路：要么做最早进圈的，要么做最晚进圈的。度假岛地图小、药多、刷圈快，进决赛圈之前，你可以一直在毒里打药。跑得快不一定赢，不跌跟头才是成功。

#### 先解决最重要的

一般来说，你在瞄远处的人的时候，身边来人了，你会先注意身边的人。

特殊情况：你和敌人在掩体的两头对峙，同时侧面的房顶有人在瞄你。你会和房顶的人对枪。

#### 打一枪换一个地方

这句话不是没有道理的。特别是在团竞地图，你把一个敌人解决了，除了他马上就复活以外，其他敌人也会知道你的方位。

如果你卡点能力特别强，一直卡同一个点——敌人也不会给你送了，或者打的是占点竞技/躲猫猫，那就没必要。

#### 近身开枪不要走位

现在还没有那个技术，二指。

这时候比的是反应速度、拉枪准确度，而不是比谁晃得快。玩音游可以锻炼反应速度，更重要的是心态和集中注意力。现在集中注意力对笔者来说很难。

俗话说穷吃鸡，富快递。装备差点，能让你更谨慎。

### 金手指、TAS 与 RTA

金手指是内存条上的金黄色导电触片。代指修改游戏内存，实现作弊。

TAS 原指 Tool-Assisted Speedrun（工具辅助竞速）。是在不修改游戏内存（但可以监控，以及利用游戏 Bug）的前提下，人**理论上**可以达到的最好成绩。

TAS 的原理是：

- 录制输入：记录下游戏每一帧的外部输入（按键/鼠标）。比如在第 300 帧，按下了【右】【A】这两个键；第 301 帧，按下了【右】【A】【B】这三个键……
- 逐帧操作：想办法让游戏程序一帧一帧地执行，我们一帧一帧地输入
- 存档/读档：如果我们对第 300 帧的输入不满意，可以重新输入。每个帧都可以存一个档，包括在这一帧之前的所有输入记录。
- 通过一系列的输入，最终让游戏加载出通关动画。这包括利用游戏 Bug，让游戏内存溢出。比如[《超级马里奥世界》](https://www.bilibili.com/video/BV1Ks411S7Fd)。

最后把游戏每一帧的外部输入记录成一个文件，这被称作录像文件（不是视频）。还包括录像作者，重录（存/读档）次数等。

对于不追求通关速度，而是表演性的 TAS 来说，S 指 Superplay。比如[超级马里奥一代[最多金币]世界记录](https://www.bilibili.com/video/BV16s41127MM)。

RTA（Real Time Attack）是真实时间通关，是不借助任何工具的实时操作，是人**实际上**的最好成绩。

## 混音

1. 有问题/有死线
2. 有扳机
3. 把软件装上

重要性 1 > 2 > 3

混音就是把好几堆数字合并，并转换为另一堆数字，使转换后的数字客观听起来好听。

声源 -> 电脑 -> 耳朵

| 声源     | 相关名词                     |
| -------- | ---------------------------- |
| 人唱歌   | 声乐                         |
| 人演奏   | 器乐、乐器、演奏             |
| 机器唱歌 | 歌声合成技术、声库、VOCALOID |
| 机器演奏 | 软件音源、编曲               |

- 人唱歌/演奏 -> 通过麦克风录制（电磁学） -> 通过声卡把模拟信号转换为数字信号 -> 通过声卡 ASIO 驱动给到电脑操作系统 -> 操作系统给到 DAW ->
- 声库/软音源 -> 给到 DAW ->
  - 存到内存里 -> 在 DAW 里用 VST 插件等进行处理（**这一步才叫混音**） ->
    - 渲染（导出、编码）文件
      - 音频文件 -> 解码器 -> 播放器 ->
      - 内存里的音频 -> DAW 实时回放 ->
        - 音频系统（ASIO、WASAPI 等）-> 输出 -> 耳朵

其中 ASIO 驱动是音频流输入输出程序， DAW（Digital Audio Workstation）是数字音频工作站，即编曲软件，也叫 VST 宿主机。VST 意思是 Vitrual Studio Technology（虚拟工作室技术），免费的 DAW 有 [Reaper](https://www.reaper.fm/) 等。

混音是一项需要非常强理性和感性的工作。如果你能做好混音，说明你心理素质非常强大。笔者现在做不到。

## 音频文件属性

把音频文件看成是一条离散采样的波形

### 采样率

**采样的时间精密度**，水平方向的。一秒钟之内采样多少个。

人耳听力范围约 $20 \sim 20000 \mathrm{Hz}$。按奈奎斯特采样定理，采样率要大于最高频率两倍，才可较准确还原。因此最常用（CD）的采样率为 $44100 \mathrm{Hz}$

或者记：记录一次振动，最少要采两个点

### 比特率

采样的空间精密度（深度），**竖直方向的**。用多少个比特切割“振幅”。

比如 $16 \mathrm {bit}$ 的位深，对于每一个采样，有 $2^{16}=65536$ 种度量“振幅”的刻度。

$1 \mathrm{bit}$ 对应 $6\mathrm{dB}$ 动态范围（？）。

## 波形统计信息

用 RX/AU 等软件可以查看音频文件的频谱图。横轴是时间，纵轴是频率，颜色亮度代表电平（振幅/能量）

用 RX 的 Waveform Statistic 查看：

| 信息                     | 单位 | 含义                                    |
| ------------------------ | ---- | --------------------------------------- |
| True peak level          | dbTP | 真峰值水平，“最高音量”                  |
| Sample peak level        | dbFS | 采样峰值水平                            |
| Max. RMS level           | db   |                                         |
| Min. RMS level           | db   |                                         |
| Total RMS level          | db   |                                         |
| Possibly clipped samples |      | 可能剪切（爆电平，超出 0dB 的样本数量） |
| DC offset                |      |                                         |
| Max.momentary loudness   |      | 最大瞬时响度                            |
| Max.short-term loudness  | LUFS | 最大短期响度                            |
| Integrated loudness      | LUFS | 整体响度                                |
| Loudness range (LRA)     | LUFS | 动态范围，情绪起伏程度                  |

## 社保是什么

社保是社会保险。再往前问一步，保险是什么？

你交钱给一个**组织**，并和他们签订**契约**。当你出事且符合**出事的条件**时，这个组织会根据契约的条文返还你对应比例的钱。

| 保险类型 | 组织     | 契约                 | 出事的条件 |
| -------- | -------- | -------------------- | ---------- |
| 商业保险 | 保险公司 | 保险合同、法律、法规 | 由契约规定 |
| 社会保险 | 国家     | 法律、法规           | 由契约规定 |

保险是一种投资，当然可以选择不交——如果你不担心：

- 老了，没钱用
- 看病，没钱用
- 工作受伤，没钱用
- 工作没了，没钱用
- 生孩子，没钱用

这五大问题的话。它们分别对应社保的：

- 基本养老保险
- 基本医疗保险
- 工伤保险
- 失业保险
- 生育保险

## 城镇职工社保和城乡居民社保

- 城镇职工社保，包括上面的五险。
- 城乡居民社保，只包括医疗保险和养老保险。
