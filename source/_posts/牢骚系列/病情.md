## 说话卡壳的原因

大脑还没有从想的模式转变成说的模式。

一个内向的人（至少包括笔者和笔者父亲）有在心里自己和自己说话的情况，这叫想的模式。

这两种模式都要经过大脑思考，如果在思考的同时把话说出来了，这叫说的模式。

说的模式的结果是：这句话我说出来了，代表这个任务完成了，大脑进入待机状态。

想的模式的结果是：这句话我说给自己听了，代表这个任务完成了，大脑进入待机状态。

考虑这样一种情况：

1. 我要使用声带震动来表达一个什么东西
2. 习惯性地使用想的模式
3. 声带没有震动，大脑判定话已经说完

打字的时候使用的是想的模式，有时小声念出来，且这时候声带是不会震动的，用气息说话。

## 为什么

我们来谈谈一个至今为止连笔者本人都没搞明白的事：为什么我要保留这个网站。既然用到 “保留” 这两个字，言外之意就是：我有把它删除的想法。为什么呢？举一个笔者小时候的亲身经历：被一个老师夸奖了，在此之后感到很羞愧。注意这里不是 “害羞”，而是 “羞愧”，如果你能明白这种感受的话。

再谈谈笔者七年前自杀的动机：注意，既然这里提到自杀了，那就说明以后不会去执行了。为什么要各种套盾，各种免责声明，这也是笔记存在的一个问题：既然写出来就是让人看的，那么你到底是写给谁看的。先下一个结论：是写给对自杀动机感兴趣的人看的，你可以把它当成一个故事，或者作为一个医疗方面的参考。再套一层盾：这不叫事后诸葛亮，在社会心理学里面有一个 “情绪的二因素理论”：1. 由生理唤起；2. 个人无法解释这种生理唤起的现象，所以必须找一个词来描述，增强确定感。

分根本原因和直接原因：

- 根本原因一：害怕人。想和人打交道，但是害怕所有人。在此之前笔者从来没有主动找人说过话，只有别人找笔者说话。在学校除了上厕所一天到晚就坐在座位上。用自杀这种行为象征性地表达 “我想让你们关注我”。
- 根本原因二：无助。在此之前看了一些网络文章讲的电子双缝干涉和光子延迟选择实验，还有缸中之脑、唯我论之类的，对这个世界抱有怀疑了，加上初中高中没有哲学课，或者是笔者没有注意听，加上后来学的是纯理科。
- 直接原因一：无助。在自杀向前推半个小时，情绪分为三个阶段：焦虑抑郁恐惧 -> 极度冷静 -> 轻度兴奋。使第一阶段发生的想法是：我非常想找人说心里话，但是我不敢，这是一个非常痛苦的过程。
- 直接原因二：对死亡报有好奇心，想知道死后的世界是什么样的，想做一个实验。或者说在下一个赌注：如果我死了，我的精神是否就自由了。这里的精神自由指的是作为一个灵魂自由地穿梭在任何场合，不会再有那些害怕的感觉。

笔者父亲曾在几年前问过笔者自杀的动机，当时只给了第二条。不给第一条是因为那时候笔者还是在处于害怕所有人的状态中。据笔者父亲回忆：从初中开始的好几年笔者都不和父母说话，他们一直不知道原因。包括第一次去找心理医生，也没有给医生说原因。那么我为什么不把“我不说话是因为我害怕你们”这句话给说出来呢。

在笔者跳完楼住院的时候，有关心笔者的人送了笔者书，有：《年轻人一定要懂的人情世故》《毛泽东妙评历史人物》《堂吉诃德》。从这些书的书名来看，因为他们不知道我是为什么自杀的，我承认是有不满的部分的。但是我必须得辩解一句：我无助的部分要远远大于不满的部分，“对社会不满”的什么玩意儿啊的想法只占 10%。

## 症状自述悖论

1. 当症状出现时，无法用语言描述自己的想法。
2. 一旦想用语言描述这个症状，开始组织语言的时候，症状就消失了。
3. 最后为了使用文字描述这个症状，不得不舍弃一些准确性，来换取一段看似合理的对症状的描述。

用函数图像打比方：语言描述就是求导，你发病的症状是魏尔斯特拉斯函数，处处连续，处处不可导：

$$
W(x) = \sum\limits_{n=0}^{\infty}a^{n}\cos(b^n \pi x) \quad a=0.5,b=13
$$

<img src="/blog/images/weierstrassa=0.5b=13.webp">

而你事后描述出来的症状是 $2\cos(\pi x)$，只是与它宏观看起来相似而已：

<img src="/blog/images/2cospix.webp">

```py
import matplotlib.pyplot as plt
import numpy as np

a = 0.5
b = 13

assert a > 0 and a < 1
assert b >= 7 and b % 2 == 1
assert (a * b) > (1 + 1.5 * np.pi)


def weierstrass(x, a, b, N=100):
    result = 0
    for n in range(N):
        result += a**n * np.cos(b**n * np.pi * x)
    return result


x = np.arange(-2, 2, 0.0001)
y1 = 2 * np.cos(np.pi * x)
y2 = weierstrass(x, a, b)

plt.figure(1)
plt.plot(x, y1, "b")
plt.figure(2)
plt.plot(x, y2, "r")

plt.show()
```

## 关于有没有真实的自我

这个问题和“一艘船换了全部零件还是不是原来的船”、“踏入几次河流”差不多。“自我”不用说，什么叫“真实”？先给笔者以某种情绪下看到的“真实”下一个定义：有这样一个人，他/她在任何场合下说出的话都是不违心的。如果由此推，“我”大概率会在不同的场合说出不同的话，或者在同一种场合以不同的情绪说出不同的话，所以根本就没有“真实的我”。在明尼苏达多相人格测试里有这样一道题目：

> 有时我也讲假话

如果你回答“否”，它会给你的“说谎”量表加一分。什么意思？这说明制定测试题目的人，默认认为做这个测试的所有人都是讲过假话的。如果你说自己从来没讲过假话，它反而会认为你在说谎。而且：“假话”的定义是什么？考虑这样一种情况：

> 我是一位心理咨询师，我遇到一位来访者。我可能对他在某些方面很有意见，但是为了保持我的工作状态，不能对他说出过激的话。

## 《和平精英》游戏操作二指转四指的心得

当你逐渐习惯四指操作后，你会发现：匹配到的对手全都会四指，该打不过还是打不过。当你高兴的时候，对面在摔手机；当你摔手机的时候，对面在高兴。这就是摔手机守恒定律。这造成一种必然的现象：你，或者你匹配到的队友，在某局游戏里会有一种异常行为，包括但不限于：开船回出生岛转一圈、驾车绕沙漠公路环行一周、把汽车喇叭或枪声打出节奏感、以及挑战不捡任何东西吃鸡。

以上都不属于“消极游戏”。属于“消极游戏”的一套操作：

1. 在超体对抗模式选“战地医师”
2. 和队友分开跳伞，在队友过不来的地方放医疗站，捡自救器
3. 捡足够多的药，准备扛毒
4. 开车找一个职业不是“闪电尖兵”的队友，和队友一起行动
5. 伺机而动，等队友进入类似这样的房子：
