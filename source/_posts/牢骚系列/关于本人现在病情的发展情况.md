## 保持清醒

最快的保持清醒的方法就是自言自语：自己和自己小声说话，把要做的事情念出来，这样注意力才不会分散。

## 症状自述悖论

“症状自述悖论”是笔者生造的一个词，指的是这样一种心理变化：

1. 当症状出现时，无法用语言描述自己的想法。
2. 一旦想用语言描述这个症状，开始组织语言的时候，症状就消失了。
3. 最后，为了使用文字描述这个症状，不得不舍弃一些准确性，来换取一段看似合理的，对症状的描述。

这个看似"合理"的描述只是事后想象出来的。而当事情真正发生的时候，是没有办法使用语言描述的。

用函数图像打比方：语言描述就是求导，你发病的症状是魏尔斯特拉斯函数，处处连续，处处不可导。

$$ 
W(x) = \sum\limits_{n=0}^{\infty}a^{n}\cos(b^n \pi x) \quad a=0.5,b=13
$$

<img src="/blog/images/weierstrassa=0.5b=13.webp">

而你事后描述出来的症状是 $2\cos(\pi x)$，只是与它宏观看起来相似而已。

<img src="/blog/images/2cospix.webp">

```py
import matplotlib.pyplot as plt
import numpy as np

a = 0.5
b = 13

assert a > 0 and a < 1
assert b >= 7 and b % 2 == 1
assert (a * b) > (1 + 1.5 * np.pi)


def weierstrass(x, a, b, N=100):
    result = 0
    for n in range(N):
        result += a**n * np.cos(b**n * np.pi * x)
    return result


x = np.arange(-2, 2, 0.0001)
y1 = 2 * np.cos(np.pi * x)
y2 = weierstrass(x, a, b)

plt.figure(1)
plt.plot(x, y1, "b")
plt.figure(2)
plt.plot(x, y2, "r")

plt.show()
```

## 关于本人现在病情的发展情况

1. 大部分时间脑子一片空白

2. 如果在想一件事的同时，听到别人说话了，且需要回应，这时回应的进程会抢在解码的进程之前出现。

可以理解为：我需要回话，但是“我需要回话”这个想法的占比比“理解听到话的意思”和“思考怎么回话”的占比都多。

3. 在打一句话还没打完的同时，突然出现打另一句话的想法了，上一句话会忘记。

4. 在电脑前面，经常忘记自己下一步要做什么。

你可以理解为注意力不集中，但这么说不准确。如果对话出现在互联网上，因为互联网是有上下文的，它不是只说一次就不能回看了，所以【我】不至于不明白【你】在说什么。

或者是在空气介质上，【我】【主动找】【他人】开一个聊天的线程，因为这时【我】是有主动权的，所以注意力会集中到听你在说什么。

又或者，当我主动把注意力转移到外部后，我能明白你在说什么。

信不信由你。

我们总是认为，所有事都得解决了才舒服。带着这些解决不了的事，照样能走下去。

## 一种自我保护的机制

这种机制最早出现于笔者高中时期，成熟于近几个星期。

最初你的摄像机是和你眼睛绑定在一起的。有一天它挪开了。

当你的心理状态切换时，脑内下层会被一股黑色的东西淹没，上表面有一台摄像机在看着：看吧，这个人居然落到了如此境地。

副作用是：笔者好像永远都哭不出来了，因为在黑色东西出现的同时，另一个人格就在看笑话。至少三年没哭了。

（在 2024 年 5 月 15 凌晨又哭了一次，因为想起初中的事情，破防了）

（父母看来：从初中开始一直就不说话）

## 关于有没有真实的自我

这个问题和“一艘船换了全部零件还是不是原来的船”、“踏入几次河流”差不多。

“自我”不用说，什么叫“真实”？先给笔者以某种情绪下看到的“真实”下一个定义：

有这样一个人，他/她在任何场合下说出的话都是不违心的。

如果由此推，“我”大概率会在不同的场合说出不同的话，或者在同一种场合以不同的情绪说出不同的话，所以根本就没有“真实的我”。

“人是一切社会关系的总和”这个命题从“呼吸、分子热运动”的角度来看太大了。

你可以由此感性地理解：大到“测量误差”，小到“测不准定理”。仪器，和使用仪器的人都不是完美的旁观者。

你可以由此感性地理解：“生气”是因为你的温度与对方的温度相差太大了，热力环流形成风。把风加上病字头，我们称之为精神病人。

## 《和平精英》游戏操作二指转四指的心得

当你逐渐习惯四指操作后，你会发现：匹配到的对手全都会四指，该打不过还是打不过。

当你高兴的时候，对面在摔手机；当你摔手机的时候，对面在高兴。这就是摔手机守恒定律。

这造成一种必然的现象：你，或者你匹配到的队友，在某局游戏里会有一种异常行为，包括但不限于：开船回出生岛转一圈、驾车绕沙漠公路环行一周、把汽车喇叭或枪声打出节奏感、以及挑战不捡任何东西吃鸡。

以上都不属于“消极游戏”。属于“消极游戏”的一套操作：

1. 在超体对抗模式选“战地医师”
2. 和队友分开跳伞，在队友过不来的地方放医疗站，捡自救器
3. 捡足够多的药，准备扛毒
4. 开车找一个职业不是“闪电尖兵”的队友，和队友一起行动
5. 伺机而动，等队友进入类似这样的房子：

<img src="/blog/images/peace-1.webp">

6. 站或蹲在这个不到一人半高的地方，把队友卡在房间里

<img src="/blog/images/peace-2.webp">

一般来说，没人会策划得这么周密。堵门这种情况在其他模式里也会出现，矛盾无外乎是队友向你要一把枪，你没给，然后队友生气了。或者是队友想和你玩一玩，拖慢你的进圈速度，因为进圈早不代表你能活下来。事实上，大部分队友要么是提前组好队了，要么是几个菜鸟以无声的默契抱团，要么是各打各的。
