就是把多个操作组合到一起。

- [官方文档](https://www.mongodb.com/docs/manual/core/aggregation-pipeline/)
- [官方文档-操作符](https://www.mongodb.com/docs/v7.0/reference/operator/aggregation-pipeline/)

`aggregate([])` 列表里可加多个 `{$操作符}`。

<!--more-->

## 按某个键的值排序输出

- `$sort:{}`

按 "\_id" 升序输出（-1 就是降序）：

```js
db.bookorder.aggregate([{ $sort: { _id: 1 } }]);
```

只是输出，并不会在原集合里排序。

对于汉字之类的大概是按 Unicode，而不是拼音。[这个网站](https://symbl.cc/cn/)可以查 Unicode。

## 输出文档个数限制

- `$limit:`

```js
db.bookorder.aggregate([{ $sort: { _id: 1 } }, { $limit: 2 }]);
```

## 跳过对前 x 个文档的操作

- `$skip:`

```js
db.bookorder.aggregate([{ $sort: { _id: 1 } }, { $skip: 3 }, { $limit: 2 }]);
```

## 按某个键分组，对各个分组的某一字段求和

```js
db.bookorder.aggregate([
  {
    $group: {
      _id: "$publishingtime",
      totalPrice: { $sum: "$price" },
    },
  },
]);
```

注意不要漏了 `$`

对非数值类的求和为 0。

## 分别进行两组操作

```js
db.goodsbaseinf.aggregate([
  {
    $facet: {
      group1: [
        { $match: { sales_vol: { $lt: 50 } } },
        { $sort: { sales_vol: -1 } },
      ],
      group2: [
        { $match: { sale_vol: { $lt: 50 } } },
        { $sort: { sale_vol: -1 } },
      ],
    },
  },
]);
```
