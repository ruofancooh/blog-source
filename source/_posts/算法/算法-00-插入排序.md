插入排序就是接扑克牌。

<!--more-->

## 介绍

插入排序就像一张一张地接扑克牌，接到第一张扑克牌时什么也不做，接到第二张扑克牌时要与第一张作比较，看插入到第一张的前面还是后面，……，接到第 n 张扑克牌时要与前 n - 1 张比较，看插入到手牌的哪个位置。

我们的手牌就是从文件读取的数据集中的每一个 Array。对于每一个 Array，每张牌就是一个元素，下标记为 `i`（index），牌面记作 `data[i]`。在接到新牌的过程中，`i` 会从 `1` 开始变大，作为外层循环的变量。

对于内层循环，就是每一次接到新牌时与旧牌作比较，然后插入某个位置。对于每一次内层循环，手牌从左到右为从小到大，且最后一张牌（新接的牌）是未经排序的，这张牌会与它的前一张牌作比较，如果比前一张牌小就交换它们的位置。

交换之后，原来最后一张牌的位置变了，相当于多了一个偏移量 `o`（offset），把 `o` 作为内层循环的变量，从 `0` 开始，则 `data[i - o]` 就是原来的最后一张牌，它与前一张牌交换位置的行为记作`[data[i - o], data[i - o - 1]] = [data[i - o - 1], data[i - o]];`。当 `data[i - o]` 比前一张牌大时，或者已经交换到了第一张的位置时，退出内层循环（当前牌插入完成），进行下一次外层循环（接新牌）。反过来说，内层循环继续的条件就是，原来的最后一张牌 `data[i - o]` 比它当前位置的前一个位置的牌 `data[i - o - 1]` 小，且没有移动到第一个位置（ `i - o != 0`，即 `i != o` ）。

## 代码

```js
function insertionSort(data) {
  const len = data.length;
  if (len < 2) {
    return;
  }
  for (let i = 1; i < len; i++) {
    for (let o = 0; o != i && data[i - o] < data[i - o - 1]; o++) {
      [data[i - o], data[i - o - 1]] = [data[i - o - 1], data[i - o]];
    }
  }
}
```
