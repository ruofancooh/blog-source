选择排序，就是拿到一手混乱的扑克牌之后，一眼盯到最小的那张，并把它和最左边的那张（下标为 0）交换位置。

然后忽略第一张，再对后面的一眼盯小，把最小的和第二张（下标为 1）交换位置。

然后忽略前两张，再次一眼盯小，和下标为 2 的交换位置……

【牢骚警告】

<!--more-->

## 介绍

程序做不到一眼盯小（假设我们不知道有算法库），所以要用一个变量 `min` 存放未排序部分的最小值。用 `mi` （min index）存放未排序部分最小值的索引，用 `fi` （first index）存放未排序部分的第一张牌的索引，用 `li` （later index）存放后面元素的索引。

对于外层循环，`fi` 从 `0` 到 `len - 2`。对于内层循环，`li` 从 `fi + 1` 到 `len - 1`。对于每一次内层循环，把当前位置的值 `data[li]` 与临时的最小值 `min` 比较，如果 `data[li] < min`，则 `min = data[li]; mi = li;`。对于临时的最小值，先取 `min = data[li]`——这时内层循环变量 `li` 应该从 `fi + 2` 开始取了，在两层循环的夹缝里令 `min = data[fi + 1]`，然后内层循环变量 `li` 从 `fi + 2` 到 `len - 1`。内层循环结束之后，再在两层循环的夹缝里交换未排序部分的最小值和未排序部分的第一张牌，即 `[data[mi], data[fi]] = [data[fi], data[mi]];`。

- 写完之后发现 `mi` 还没定义，它是用来存放未排序部分最小值的索引的，所以应该在两层循环的夹缝里令 `mi = fi + 1`。

- 运行之后发现错了，应该在夹缝里令 `min = data[fi]; mi = fi;`，内层循环一开始令 `li = fi + 1`。再次运行之后成功了。

## 代码

```js
function selectionSort(data) {
  const len = data.length;
  for (let fi = 0; fi < len - 1; fi++) {
    let min = data[fi];
    let mi = fi;
    for (let li = fi + 1; li < len; li++) {
      if (data[li] < min) {
        min = data[li];
        mi = li;
      }
    }
    [data[mi], data[fi]] = [data[fi], data[mi]];
  }
}
```
