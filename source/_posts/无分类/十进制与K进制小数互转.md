## K 转 10：和整数进制转换的方法一样

每一位分别乘上 K 的次方，再相加。

### 2 转 10

#### 有限小数

$11.1011_{2} = 3.6875_{10}$

| 项              | 值       |
| --------------- | -------- |
| $1\times2^{1}$  | $2$      |
| $1\times2^{0}$  | $1$      |
| $1\times2^{-1}$ | $0.5$    |
| $0\times2^{-2}$ | $0$      |
| $1\times2^{-3}$ | $0.125$  |
| $1\times2^{-4}$ | $0.0625$ |
| **总和**        | $3.6875$ |

#### 无限循环小数

$1.1 \dot{0} 11 \dot{0} ... _{2} = 1.7_{10}$

可以转换成分数计算：

$$
\begin{aligned}
1.1 \dot{0} 11 \dot{0} ... _{2} & = x \\
11.\dot{0} 11 \dot{0} ... _{2} & = 2_{10}x \\
110110. \dot{0} 11 \dot{0} ... _{2} & = 32_{10}x \\
两式相减，110011_{2} & = 30_{10}x \\
x & = \frac{110011_{2}}{30_{10}} = \frac{51_{10}}{30_{10}} = 1.7_{10}
\end{aligned}
$$

## 10 转 K：乘 K 取整

整数部分仍然除 K 取余，小数部分乘 K 取整，然后相加。

### 10 转 2

$0.7 _{10} = 0.1 \dot{0} 11 \dot{0} ... _{2}$

| 小数部分乘 2         | 取整数部分 |
| -------------------- | ---------- |
| $0.7 \times 2 = 1.4$ | $1$        |
| $0.4 \times 2 = 0.8$ | $0$        |
| $0.8 \times 2 = 1.6$ | $1$        |
| $0.6 \times 2 = 1.2$ | $1$        |
| $0.2 \times 2 = 0.4$ | $0$        |
| ...                  | ...        |

### 10 转 16

$0.7 _{10} = 0.\mathrm{B} \dot{3} ... _{16}$

| 小数部分乘 16          | 取整数部分   |
| ---------------------- | ------------ |
| $0.7 \times 16 = 11.2$ | $\mathrm{B}$ |
| $0.2 \times 16 = 3.2$  | $3$          |
| ...                    | ...          |
