这里的【照片作业】指的是用手机拍摄写在纸上的字。具体步骤：

1. 拍摄
2. 裁剪
3. **彩色转黑白**
4. **白平衡**
5. 导出图片
6. （可选）导入 Word，导出 PDF

<!--more-->

1. 首先要拍清晰，手不抖。
   <img src="/blog/images/hwimg-source.jpg">
   （不太清晰）
2. 照片中有用的部分是什么？是作业内容（文字），和反衬内容的背景（纸）。在拍完之后适当裁剪。
3. 同上，老师只需要分辨字和背景，所以照片完全可以转成黑白的。我使用的是[GIMP](https://www.gimp.org/downloads/)，是开源免费的。用手机自带的修改也可。
   - GIMP -> 文件 -> 打开 -> 选择图片
   - GIMP -> 颜色 -> 去色 -> 彩色到灰度
     <img src="/blog/images/hwimg-c2g.jpg">

看起来雾蒙蒙的，而且太白了。

4. 白平衡

   - GIMP -> 颜色 -> 自动 -> 白平衡
     <img src="/blog/images/hwimg-wb.jpg">
     看起来好多了。

5. 导出图片

   - GIMP -> 文件 -> 导出为 -> 文件名.jpg -> 导出
   - 其中，【保存 Exif 数据】、【保存 XMP 数据】、【保存 IPTC 数据】、【保存缩略图】、【保存色彩配置文件】都不用选。【高级选项】调了没看出明显区别，所以不用调。
   - 导出质量设成 `10` 就行，质量越低文件越小，只要还能分辨出字就行。
   - 我用 `10` 导出看起来和导出前没啥区别，但是用 `0` 导出就：
     <img src="/blog/images/hwimg-0.jpg">
     有点恐怖。~~但是更清楚了~~

6. 导入 Word，转成 PDF

   - 前提是老师允许交文档。好处是上下滑动查看连续的，比左右滑动查看单张的更人性化。
   - Word -> 布局 -> 页边距 -> 四个边距都设成 `0`。
   - Word -> 插入 -> 图片 -> 此设备 -> 按住 `Ctrl` 点选多张图片。注意第一个选的会被插在最后，是反着来的。
   - 把图片拉伸到与页面等宽，尽可能每一页不留太多空白。
   - Word -> 文件 -> 导出 -> 创建 PDF/XPS 文档。
   - 导出选项有【标准】和【最小文件大小】。先选最小的，看清不清楚，能看就行。就是用有损压缩牺牲了部分清晰度，换取了比原图和 Word 文档都小的体积。四五张照片原图有上十 MB，用`10`质量导出有 2MB 左右，再转成 PDF 能在 1MB 之内。
   - PDF 还有个好处就是跨平台兼容，各个平台版式显示一致。
   - 不允许交文档时，或许可以交拼图。但是太长了不好看，宽度也不好调，不像文档有分页。
